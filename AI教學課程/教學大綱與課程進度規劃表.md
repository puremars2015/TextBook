## 課程詳細內容

### 第1堂｜7/21 — 【AI原生開發與LLM核心架構】（學科）

**授課時間：** 18:30-21:30（3小時）

1. 大語言模型LLM產業應用
2. AI Agent系統架構
3. 如何用AI增加你的個人工作生產力
4. Vibe Coding技術限制、長處與產業應用邊界

> **說明：** 本課程會先由教師展示如何用 Vibe Coding 的技巧，直接操作 GitHub Copilot 來生成一個最基礎的 Agent 機制，展示 Agent 概念，後續課程會讓學員嘗試自己創建 Agent，也由此讓學員知道 OpenClaw 的概念，以及因為機制設計與 LLM 的特性所造成的資安風險原因，與可能的應對手段。

---

### 第2堂｜7/28 — 【AI Agent邏輯建構與自動化引擎配置】（術科）

**授課時間：** 18:30-21:30（3小時）

1. AI Agent 流程自動化工具之核心架構與底層邏輯設計
2. 進階控制流實務：掌握布林邏輯條件判斷與迴圈演算
3. 自動化流程異常監控：模擬並構建錯誤情境之容錯補償機制
4. 使用 Coding Agent（GitHub Copilot）與 Python 寫一個基本的與 LLM 對話的機制，然後將該機制嫁接上特定工具調用檔案編輯功能的程式，來說明 Agent 的概念

---

### 第3堂｜8/4 — 【巨量數據調度工程與非結構化資料轉換】（術科）

**授課時間：** 18:30-21:30（3小時）

1. AI Agent 於海量資料處理之典型場景分析與系統架構
2. 數據表變數動態管理：執行複雜資料庫之篩選、排序與寫入邏輯
3. 跨活頁簿數據整合工程：建立批次處理流程，自動化彙整異質報價與財務報表

---

### 第4堂｜8/11 — 【跨平台Web數據擷取與異質系統整合】（術科）

**授課時間：** 18:30-21:30（3小時）

1. Web UI 自動化技術：網頁動態表單映射、非結構化資料擷取與認證流程設計
2. 實戰演練：自動化登錄特定平台並執行數據探勘與結構化存儲
3. 承接前一堂課的概念，這次我們導入 Nvidia 的 NemoClaw，透過 Docker 安裝 NemoClaw，來完成相對應的任務處理
