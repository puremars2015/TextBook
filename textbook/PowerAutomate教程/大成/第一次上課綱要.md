# 第一週上課綱要

## 課程目標

1. 基本掌握 Power Automate 用途說明與操作介面。 Cloud Flow跟Desktop Flow的差異。
2. 基本Power Automate Desktop操作與流程建立。
- 能了解桌面流程的流程、變數、條件、迴圈等基本概念。
- 能建立簡單的桌面自動化流程，達成重複性工作自動化。
3. 實作常見場景的自動化流程。
- 自動檔案歸檔練習

## 預計課程Schedule

* 14:00-14:20 Power Automate介紹，Cloud Flow + Desktop Flow差異說明
* 14:20-16:00 Power Automate Desktop入門教學
    - 基本介面介紹
    - 流程建立與執行
    - 變數與資料型態
    - 條件判斷與迴圈
    - 子流程建立與應用,全域與非全域變數說明
    - 動作錄影功能介紹與應用
* 16:00-17:00 自動檔案歸檔Flow開發練習

note. 以上時間可依現場狀況彈性調整,休息時間穿插其中,估計約為每小時休息10分鐘

## 課程內容大綱（詳細步驟與練習項目）

- 導入與學習目標回顧（10 分）
    - 說明本週學習重點與實作目標：熟悉 Power Automate Desktop（PAD）介面、建立可執行的桌面流程、完成自動檔案歸檔實作。

- Power Automate 基本概念（15 分）
    - Cloud Flow 與 Desktop Flow 差異、適用場景與限制。
    - PAD 架構概覽：流程（flow）、動作（actions）、觸發與輸出。

- PAD 介面與基本操作（25 分）
    - 主畫面介紹：流程清單、動作面板、屬性窗格、偵錯/執行面板。
    - 建立與執行第一個簡單流程（開啟應用程式、讀取檔案清單）。

- 變數與資料型態（20 分）
    - 變數類型（文字、數字、清單、資料表）與宣告方式。
    - 存取與傳遞變數、範例：用變數暫存檔名、路徑。

- 條件判斷與迴圈（30 分）
    - If / Else 判斷：根據副檔名或日期分支處理。
    - 迴圈（For each、While）用法：遍歷檔案清單、批次處理。
    - 例外處理基礎示範。

- 子流程（Subflow）與變數作用域（15 分）
    - 建立子流程、參數傳遞、全域與區域變數差異。
    - 範例：將「檔案解析」拆成子流程以重用。

- 動作錄影功能（15 分）
    - 錄製桌面動作（Recorder）取得滑鼠與鍵盤操作，與流程整合與調整。
    - 錄影後的操作最佳化（替換硬編碼路徑為變數、加入等待條件）。

- 實作練習：自動檔案歸檔（60 分）
    - 練習目標：建立一個可運行的桌面流程，定期掃描來源資料夾，依條件分類並移動檔案到目標資料夾。
    - 步驟要點：
        1. 讀取來源資料夾檔案清單（支援子目錄與篩選副檔名）。
        2. 對每個檔案解析檔名或讀取檔案屬性（例如：建立日期、關鍵字）。
        3. 判斷目標分類（依條件決定目標資料夾路徑）。
        4. 若目標資料夾不存在則建立，然後移動檔案。
        5. 錯誤捕捉：處理檔案存取衝突與記錄失敗項目。
        6. 完成後產生簡易報表或日誌（CSV 或 Excel）。
    - 驗收標準：流程可執行、正確分類至少三種情境、並有錯誤紀錄。

- 測試、偵錯與效能建議（10 分）
    - 使用斷點、日誌輸出、逐步執行檢查變數內容。
    - 避免硬編碼路徑、適當加入等待/重試機制。

- 問題討論與 Q&A（10-15 分）
    - 回顧常見錯誤與解法、現場 Q&A、安排下一課預習內容。
