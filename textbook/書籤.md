MCP 聚合網站與伺服器資源平台整理
 1. MCP.so
 • 網址：https://mcp.so
 • 介紹：由 Anthropic 開發的開源協議，旨在讓 AI 系統（如 Claude）安全地連接各種資料來源。
 2. 阿里雲百煉
 • 網址：https://www.aliyun.com/product/bailian
 • 介紹：阿里雲推出的企業級大模型開發平台，提供多模態模型調用服務，助力企業打造 AI 應用。
 3. HiMCP
 • 網址：https://himcp.ai
 • 介紹：一個 AI 驅動的工具，能夠從自然語言描述生成現代化的 UI 元件，並與主流 IDE 整合，簡化 UI 開發流程。
 4. Smithery.ai
 • 網址：https://smithery.ai
 • 介紹：MCP 伺服器的註冊與管理平台，提供超過 2000 個 MCP 伺服器的索引，方便用戶發現、安裝和管理。
 5. shareMCP
 • 網址：https://sharemcp.com
 • 介紹：專注於 Model Context Protocol (MCP) 的導航站點，提供豐富的 MCP 相關資源、工具和服務的分類展示。
 6. MCP Market
 • 網址：https://mcpmarket.com
 • 介紹：一個 MCP 伺服器的市場，連接 MCP 客戶端（如 Claude 和 Cursor）到各種工具，如 Figma、Databricks 等。
 7. PulseMCP
 • 網址：https://www.pulsemcp.com
 • 介紹：提供 MCP 用例、伺服器、客戶端和新聞的瀏覽與發現，是 MCP 生態系統的社群資源。
 8. Glama MCP
 • 網址：https://glama.ai/mcp/servers
 • 介紹：提供開源的 MCP 伺服器實作，支援 AI 助手透過 ScreenshotOne API 擷取網站截圖。
 9. cursor.directory
 • 網址：https://cursor.directory
 • 介紹：Cursor 社群的聚集地，提供探索和生成規則、瀏覽 MCPs、發布和關注最新消息、學習、連接和發現工作機會等功能。
 10. Portkey.ai
 • 網址：https://portkey.ai
 • 介紹：提供 AI Gateway、Prompts、Guardrails 和 Observability Suite，協助團隊快速部署可靠、成本效益高且快速的應用程式。
 11. Cline’s MCP Marketplace
 • 網址：https://cline.bot/mcp-marketplace
 • 介紹：探索 Model Context Protocol (MCP) 市場，這裡收集了 Cline 上可用的強大工具和功能。
 12. MCP Hub
 • 網址：https://mcp.so/server/mcp-hub
 • 介紹：MCP Hub 是一個全面的框架，用於構建、管理和部署 MCP 客戶端和伺服器。提供工具與資源協助開發人員整合 MCP。
 13. MCPServers
 • 網址：https://mcp.so/server/mcp-servers-hub
 • 介紹：此存儲庫為集中式目錄，提供開源與專有 MCP 伺服器的列表，包括功能、文檔鏈接與貢獻者資訊。
 14. Awesome MCP Servers
 • 網址：https://mcpservers.org
 • 介紹：收錄各種 MCP 伺服器的集合，包括與 Google News、MongoDB、Notion 等的整合。
